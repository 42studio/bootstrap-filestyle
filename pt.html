jQuery FileStyle para Bootstrap

Personalize input file html para Bootstrap Twitter com jQuery

Projeto no Github Download Filestyle

Tweet

Descrição

O Filestyle Bootstrap é um plugin jquery para personalizar input file usando o Twitter Bootstrap.

Este plugin possui uma interessante forma de se apresentar um formulário para envio de arquivos de maneira atraente, sem deixar de ser simples e eficiente.

Requisitos

Twitter Bootstrap

jQuery

Instalação

Inclua o arquivo bootstrap-filestyle.min.js no seu código html.

<script type="text/javascript" src="js/bootstrap-filestyle.min.js"> </script>

Usando

Uso básico:

$(":file").filestyle()

Usando opções:

$(":file").filestyle({
      textField: "My files",
      icon: true
})

Usando método:

$(":file").filestyle("clear")

Opções

buttonText String Personalização do texto do botão.
textField Bool Mostra ou esconde o campo de texto.
classButton String Use classes Bootstrap no botão.
classText String Use classes Bootstrap para o campo de texto.
icon Bool Adiciona um icone no botão.
classIcon String Use of classes Bootstrap para o icone.

Métodos

clear String Limpa um filestyle definido.
